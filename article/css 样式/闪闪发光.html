CSS Logo 闪光效果

实现闪光效果的CSS代码：

/* 创建闪光效果的关键代码 */
.logo::before {
    content: "";
    position: absolute;
    top: -50%;
    left: -50%;
    width: 200%;
    height: 200%;
    background: linear-gradient(
        to bottom right,
        transparent,
        transparent,
        rgba(255, 255, 255, 0.1),
        rgba(255, 255, 255, 0.4),
        rgba(255, 255, 255, 0.1),
        transparent,
        transparent
    );
    transform: rotate(45deg);
    animation: shine 3s infinite;
}

@keyframes shine {
    0% {
        transform: rotate(45deg) translateX(-100%);
    }
    20% {
        transform: rotate(45deg) translateX(100%);
    }
    100% {
        transform: rotate(45deg) translateX(100%);
    }
}

实现原理：

这种闪闪发光的效果是通过创建一个斜向的线性渐变层，并使用动画让它从左上角移动到右下角实现的：

1. 使用::before伪元素创建一个覆盖整个Logo的层

2. 使用linear-gradient创建一个透明到白色再到透明的渐变效果

3. 将渐变层旋转45度使其呈对角线方向

4. 应用关键帧动画shine，让这个渐变层从左上角(-100%)移动到右下角(100%)

5. 通过调整动画时间和渐变参数，可以创建出不同的闪光效果
